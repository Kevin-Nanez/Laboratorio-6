¿Que es el Modal?
El modal de Bootstrap es un componente de la biblioteca de CSS y JavaScript llamada Bootstrap, que se utiliza para mostrar contenido en una ventana emergente que se superpone en la página web principal. Es un cuadro de diálogo que aparece en la parte superior de la página, centrado en la pantalla, y se utiliza comúnmente para mostrar información adicional, solicitar la entrada del usuario o confirmar una acción.
